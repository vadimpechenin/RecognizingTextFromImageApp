I. Вернуться на нужный коммит:

git reset --hard <номер коммита>


II. Для новой ветки

1. Вернуться на нужный коммит.

2. Создать и перейти на новую ветку

 git checkout -b <имяветки>

или 

git branch <имяветки>
git checkout <имяветки>

Или же 

2. Создать изменения, git add ., git commit -m "<>"

3. git branch <имяветки>
git checkout <имяветки>

III. Удалить последний коммит
1. git reset --hard HEAD~
2. Запушить на удаленный репозиторий: git push origin main -force
3. Или действовать в меню: git rebase -i HEAD~2
